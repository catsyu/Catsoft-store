Tutorial LUT iPhone

LUT Apple Log ProRes

Panduan ini untuk customer yang ingin memakai 4 file LUT Catsoft pada footage Apple Log dari iPhone Pro. Pastikan video direkam dalam Apple Log agar warna LUT tampil sesuai.

Langkah Awal

Siapkan File Dan Footage Apple Log

  1. Simpan file LUT di iPhone

    Download file dari admin Catsoft, lalu simpan atau ekstrak file ke aplikasi Files. Pastikan file yang akan dipakai berformat .cube.

  2. Aktifkan Apple ProRes Log

    Di iPhone, buka Settings > Camera > Formats, aktifkan Apple ProRes. Saat merekam di Camera bawaan, pilih mode Video lalu pilih ProRes Log jika tersedia.

  3. Cek ruang penyimpanan

    Video ProRes/Log ukurannya besar. Sebelum merekam, pastikan storage iPhone cukup atau gunakan external storage yang mendukung jika Anda merekam durasi panjang.

  4. Pilih Workflow

    Gunakan Blackmagic Camera untuk preview LUT saat merekam, Kino untuk grade langsung di iPhone, atau aplikasi editing seperti DaVinci Resolve, Final Cut Pro, dan CapCut desktop untuk proses setelah shooting.

  5. Simpan File Asli

    Simpan video Apple Log original dan file LUT di folder yang mudah ditemukan. Jangan hapus file asli sebelum hasil edit final selesai diexport.

Kompatibilitas

Khusus Pengguna iPhone Apple Log

Produk ini dioptimalkan untuk iPhone 15 Pro, iPhone 15 Pro Max, iPhone 16 Pro, dan iPhone 16 Pro Max. Berdasarkan dokumentasi Apple, Log recording membutuhkan iPhone 15 Pro atau yang lebih baru, sehingga model Pro lebih baru yang mendukung Apple Log juga bisa digunakan.

Tutorial Per Aplikasi

Blackmagic Camera

Import Dan Preview LUT

  1. Buka file LUT dari Files

    Simpan semua file .cube di folder Files yang mudah ditemukan, misalnya Catsoft LUT Apple Log.

  2. Buka Blackmagic Camera

    Masuk ke menu Settings atau pengaturan LUT di Blackmagic Camera, lalu cari bagian LUTS.

  3. Pilih Import LUT

    Tekan Import LUT, pilih file .cube dari Files, lalu tunggu sampai nama LUT muncul di daftar LUT.

  4. Aktifkan Display 3D LUT

    Gunakan Display 3D LUT jika ingin melihat preview warna saat merekam. Ini membantu menilai tone tanpa harus menunggu proses edit.

  5. Tentukan apakah LUT ikut direkam

    Jika tersedia opsi Record LUT to Clip, aktifkan hanya bila ingin look LUT menempel pada file. Untuk editing fleksibel, cukup gunakan LUT sebagai preview dan simpan footage Apple Log original.

Rekomendasi

Jangan Buru-Buru Bake LUT

Untuk hasil paling aman, rekam Apple Log original lalu gunakan LUT saat editing. Preview LUT boleh aktif agar warna di layar terlihat mendekati hasil akhir.

Lihat Kino
Kino App

Import LUT ke Kino

  1. Simpan file .cube di Files

    Pastikan file LUT sudah diekstrak dan terlihat sebagai file .cube, bukan masih dalam ZIP/RAR.

  2. Buka dari Files

    Tap file .cube, pilih Share atau Open In, lalu pilih Kino. Jika opsi tidak muncul, buka Kino terlebih dahulu lalu masuk ke Settings > Grade > Import.

  3. Pilih input Apple Log

    Saat import, pilih input Apple Log agar Kino membaca LUT sesuai jenis footage yang direkam.

  4. Simpan sebagai grade

    Beri nama grade sesuai file LUT Catsoft, lalu simpan. Ulangi proses untuk LUT lain yang ingin digunakan.

  5. Tes rekaman pendek

    Rekam 5-10 detik sebagai test, cek warna kulit, highlight, dan shadow. Jika terlalu kuat, pilih LUT lain atau koreksi exposure sebelum rekaman utama.

Penting

Input Harus Apple Log

Jika input dipilih salah, warna bisa terlalu kontras atau tidak natural. Untuk LUT Catsoft ini, pilih Apple Log saat import di Kino.

Lihat Editing
Editing Desktop

Pakai LUT saat edit video

  1. Pindahkan footage dan LUT ke komputer

    Kirim video Apple Log dan file .cube ke Mac/Windows. Simpan dalam satu folder project agar mudah dicari.

  2. Import video ke aplikasi editing

    Buka DaVinci Resolve, Final Cut Pro, CapCut desktop, atau aplikasi lain yang mendukung LUT .cube, lalu import footage Apple Log.

  3. Apply LUT ke clip atau adjustment layer

    Gunakan menu LUT/Color/Effects sesuai aplikasi. Pilih file .cube Catsoft, lalu apply ke clip utama atau adjustment layer agar mudah dikontrol.

  4. Koreksi exposure dan white balance

    Jika hasil terlalu terang, gelap, kuning, atau biru, koreksi exposure dan white balance sebelum atau setelah LUT sampai warna terlihat natural.

  5. Export Hasil Akhir

    Setelah warna aman, export ke format yang dibutuhkan. Untuk upload social media, gunakan resolusi dan bitrate sesuai platform tujuan.

Catatan Aplikasi

Menu LUT tiap editor berbeda

Nama menunya bisa berbeda di tiap aplikasi. Yang penting aplikasi tersebut mendukung import LUT .cube. Jika tidak menemukan menu LUT, kirim screenshot aplikasi ke admin.

Tips Dan Kendala

Checklist Hasil Rapi

Gunakan Apple Log

Jangan apply LUT ini ke video standar non-Log jika ingin warna tetap natural.

Jaga Exposure

Hindari highlight terlalu putih dan bayangan terlalu gelap agar detail tetap bisa diproses.

Kunci White Balance

White balance yang berubah-ubah membuat warna tiap clip tidak konsisten.

Simpan File Asli

Backup file LUT dan footage original sebelum edit agar aman saat revisi.
