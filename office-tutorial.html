Microsoft Office 2024 Pro Plus

Ikuti langkah berikut setelah menerima link installer dan license key dari admin Catsoft. Pastikan koneksi internet aktif saat proses instalasi dan aktivasi.

Mulai Tutorial Cek Kendala
Yang perlu disiapkan
  • PC atau laptop Windows 10/11
  • Installer Office dari admin
  • License key Office
  • Koneksi internet aktif
Langkah utama

Instalasi dan aktivasi

  1. Download installer Office

    Buka link installer Office 2024 Pro Plus, lalu download file sampai selesai.

    Download Office
  2. Jalankan file installer

    Buka file yang sudah didownload. Jika muncul izin dari Windows, pilih Yes, lalu tunggu proses instalasi berjalan.

  3. Buka aplikasi Office

    Setelah instalasi selesai, buka salah satu aplikasi seperti Word, Excel, atau PowerPoint dari Start Menu.

  4. Masukkan license key

    Di Word, Excel, atau PowerPoint, pilih opsi aktivasi dengan product key, lalu masukkan kode lisensi yang diberikan admin.

  5. Pilih Activate by telephone

    Jika muncul Activation Wizard, pilih metode I want to activate the software by telephone atau Activate by telephone, lalu lanjutkan.

  6. Kirim screenshot Installation ID

    Ambil screenshot bagian Installation ID yang muncul di Activation Wizard, lalu kirim ke admin Catsoft agar kami bantu proses sampai mendapatkan Confirmation ID.

  7. Masukkan Confirmation ID

    Setelah admin mengirimkan Confirmation ID, masukkan kode tersebut di Activation Wizard, lanjutkan, lalu buka menu Account untuk memastikan Office sudah aktif.

Catatan penting

Jika muncul Activation Wizard

Pilih Activate by telephone, jangan tutup halaman wizard, lalu kirim screenshot Installation ID ke admin Catsoft. Kami akan bantu kirimkan Confirmation ID untuk menyelesaikan aktivasi.

Troubleshooting

Kendala yang sering terjadi

Installer tidak terbuka

Pastikan file sudah selesai didownload. Jika masih berupa arsip, ekstrak terlebih dahulu, lalu jalankan file setup.

Office lama masih terpasang

Uninstall Office lama terlebih dahulu jika instalasi baru gagal atau aplikasi masih membuka versi sebelumnya.

Key tidak diterima

Cek ulang huruf dan angka pada key. Jika tetap gagal, kirim screenshot error ke admin Catsoft.

Diminta aktivasi telepon

Pilih Activate by telephone, screenshot Installation ID, lalu kirim ke admin untuk dibantu mendapatkan Confirmation ID.
