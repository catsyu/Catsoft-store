Tutorial Preset Foto

Preset Lightroom 140++ Street Photography

Panduan ini membantu customer memasang preset Catsoft di Lightroom mobile dan desktop. Pilih langkah sesuai perangkat yang digunakan agar file tidak salah masuk.

Pilih Metode Mulai dari Awal
Yang perlu disiapkan
  • Aplikasi Adobe Lightroom sudah terpasang
  • File preset dari admin Catsoft
  • File sudah diekstrak jika masih ZIP/RAR
  • Foto yang ingin diedit
Langkah awal

Siapkan file preset

  1. Download file dari admin

    Simpan file Preset Lightroom 140++ dari admin Catsoft ke perangkat Anda. Jangan rename folder utama sebelum proses pemasangan selesai.

  2. Cek format file

    Jika file berupa ZIP/RAR, ekstrak terlebih dahulu. Biasanya preset tersedia dalam format XMP/ZIP untuk import langsung atau DNG untuk metode mobile.

  3. Update aplikasi Lightroom

    Pastikan Adobe Lightroom di HP, iPad, Mac, atau Windows sudah versi terbaru agar menu preset dan fitur import lebih mudah ditemukan.

  4. Pilih metode sesuai device

    Gunakan metode Android, iPhone/iPad, Lightroom Desktop, atau Lightroom Classic sesuai aplikasi yang Anda pakai. Jika ragu, kirim screenshot tampilan Lightroom ke admin.

Catatan penting

Hasil warna bisa berbeda

Preset bekerja mengikuti kondisi foto. Jika hasil terlalu gelap, terang, atau terlalu kuat, sesuaikan Exposure, White Balance, Contrast, dan Preset Amount setelah preset diterapkan.

Pilih metode

Tutorial pemasangan preset

Metode 1

Import XMP/ZIP di Lightroom mobile

  1. Simpan file preset di HP/iPad

    Download file ZIP/XMP dari admin Catsoft. Jika file berbentuk ZIP dan Lightroom bisa memilih ZIP, file tidak wajib diekstrak.

  2. Buka Lightroom mobile

    Buka aplikasi Lightroom, lalu buka salah satu foto. Menu preset biasanya muncul setelah foto dibuka di mode edit.

  3. Masuk ke Presets

    Tap Presets, lalu tekan menu tiga titik di area panel Presets. Pilih Import Presets jika menu tersebut tersedia.

  4. Pilih file ZIP/XMP

    Pilih file preset dari Files, Gallery, atau folder download. Tunggu sampai proses import selesai.

  5. Cek folder preset

    Buka Presets > Yours > User Presets atau nama group preset Catsoft. Jika belum muncul, tutup dan buka lagi Lightroom.

Jika menu tidak ada

Gunakan DNG atau sync desktop

Pada beberapa versi perangkat, menu Import Presets bisa berbeda atau tidak tampil. Jika begitu, pakai metode DNG mobile atau import dari Lightroom Desktop lalu sync ke mobile.

Metode DNG
Metode 2

Buat preset dari file DNG

  1. Ekstrak file DNG

    Jika preset dikirim dalam ZIP/RAR, ekstrak dulu sampai file DNG terlihat. Simpan di Files, Photos, Gallery, atau cloud storage yang mudah dibuka.

  2. Import DNG ke Lightroom

    Buka Lightroom mobile, tap Add Photos. Di iPhone/iPad pilih From Camera Roll atau From Files. Di Android pilih Gallery/Folder atau lokasi cloud storage.

  3. Buka foto DNG

    Jika muncul tampilan abu-abu atau pesan file tidak bisa diload, lanjutkan saja selama file DNG berhasil masuk ke Lightroom.

  4. Pilih Create Preset

    Tap menu tiga titik di kanan atas, pilih Create Preset, beri nama preset sesuai nama file, lalu pilih atau buat Preset Group Catsoft Street.

  5. Simpan dan ulangi

    Tekan tanda centang/simpan. Ulangi langkah yang sama untuk setiap file DNG yang ingin dijadikan preset.

Nama preset

Buat group khusus

Agar mudah dicari, gunakan satu group seperti Catsoft Street 140++. Jangan mencampur preset baru dengan preset lain jika ingin katalog tetap rapi.

Cara pakai
Metode 3

Import di Lightroom Desktop

  1. Simpan file di komputer

    Download file preset dari admin Catsoft ke Mac/Windows. Jika file masih ZIP, Anda bisa coba import ZIP langsung atau ekstrak terlebih dahulu jika dibutuhkan.

  2. Buka Lightroom Desktop

    Gunakan aplikasi Lightroom, bukan Lightroom Classic, jika ingin preset lebih mudah sync ke mobile dengan akun Adobe yang sama.

  3. Pilih Import Profiles & Presets

    Dari menu bar, pilih File > Import Profiles & Presets, lalu pilih file ZIP/XMP yang diberikan.

  4. Klik Import

    Tunggu proses import selesai. Preset biasanya muncul di panel Presets, bagian Yours, User Presets, atau nama group preset.

  5. Cek sync mobile

    Jika login dengan akun Adobe yang sama, preset desktop dapat tersinkron ke Lightroom mobile. Pastikan internet aktif dan sync tidak dijeda.

Paling praktis

Desktop lalu sync

Untuk pengguna yang punya Mac/Windows, metode desktop sering paling rapi karena preset dapat masuk sekaligus lalu muncul di mobile melalui sync Adobe.

Metode 4

Import di Lightroom Classic

  1. Buka module Develop

    Import preset dilakukan dari module Develop. Buka salah satu foto agar panel Presets terlihat.

  2. Cari panel Presets

    Di panel Presets, tekan tombol plus atau menu preset, lalu pilih Import Presets.

  3. Pilih file ZIP/XMP

    Pilih file preset dari folder download. Jika ZIP tidak terbaca, ekstrak ZIP lalu import file XMP di dalamnya.

  4. Cek User Presets

    Setelah import, cek User Presets atau group preset Catsoft. Jika belum muncul, restart Lightroom Classic.

  5. Gunakan pada foto

    Pilih foto, klik preset, lalu sesuaikan exposure, white balance, dan contrast agar hasil tidak terlalu kuat.

Classic berbeda

Bukan Lightroom Desktop

Lightroom Classic dan Lightroom Desktop tampilannya berbeda. Jika Anda ingin preset muncul di HP, lebih mudah import lewat Lightroom Desktop atau gunakan metode DNG di mobile.

Lihat Desktop
Metode 5

Sync preset antar device

  1. Login akun Adobe yang sama

    Pastikan Lightroom di komputer dan Lightroom mobile memakai akun Adobe yang sama.

  2. Import preset dari desktop

    Gunakan metode Lightroom Desktop, lalu tunggu sampai preset muncul di panel Presets.

  3. Aktifkan internet dan sync

    Biarkan Lightroom terbuka beberapa menit. Jangan logout atau menutup aplikasi saat proses sync belum selesai.

  4. Buka Lightroom mobile

    Di HP/iPad, buka Lightroom mobile, buka foto, lalu cek Presets > Yours. Preset akan muncul jika sync selesai.

  5. Jika belum muncul

    Restart Lightroom di kedua device, cek akun Adobe, dan pastikan storage/cloud sync tidak penuh atau paused.

Tanpa sync

Import manual per device

Jika tidak menggunakan sync Adobe, pasang preset manual di tiap perangkat menggunakan metode XMP/ZIP atau DNG.

Kembali ke Mobile
Setelah terpasang

Cara pakai preset

  1. Buka foto yang ingin diedit

    Pilih foto street, outdoor, indoor, atau foto harian yang ingin diberi tone. Preset akan lebih rapi jika foto tidak terlalu gelap atau overexposure.

  2. Pilih Presets

    Buka panel Presets, masuk ke folder preset Catsoft atau User Presets, lalu pilih tone yang paling cocok dengan foto.

  3. Atur intensitas

    Jika Lightroom Anda memiliki slider Amount, turunkan atau naikkan intensitas preset. Jika tidak ada, atur manual Exposure, Shadows, Highlights, Temperature, dan Tint.

  4. Simpan hasil edit

    Jika hasil sudah sesuai, export foto ke galeri atau folder tujuan. Simpan juga file preset asli agar bisa dipasang lagi jika ganti perangkat.

Sinkron antar device

Gunakan akun Adobe yang sama

Jika preset dipasang dari Lightroom Desktop dan ingin muncul di mobile, login dengan akun Adobe yang sama dan pastikan sinkronisasi cloud aktif. Jika tidak memakai sync, pasang preset manual di tiap device.

Troubleshooting

Kendala yang sering terjadi

Preset tidak muncul

Tutup dan buka lagi Lightroom, lalu cek folder Yours, User Presets, atau nama group preset. Pastikan file yang dipilih benar, bukan folder kosong.

ZIP tidak bisa dipilih

Coba ekstrak ZIP, lalu pilih file XMP di dalamnya. Jika di mobile tetap tidak bisa, gunakan metode DNG dan buat preset manual dari file contoh.

Warna terlalu kuat

Turunkan Amount jika tersedia, atau sesuaikan Exposure, Contrast, Saturation, Highlights, dan Shadows sampai warna lebih natural.

Tidak sinkron ke HP

Pastikan login dengan akun Adobe yang sama, koneksi internet aktif, dan sync tidak dijeda. Jika tidak memakai sync, import ulang preset di HP.
