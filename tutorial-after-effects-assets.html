Pilih workflow

Tutorial sesuai kebutuhan

Windows

Cara menggunakan assets di PC Windows

  1. Download dan ekstrak file

    Simpan file dari admin Catsoft, lalu ekstrak ZIP/RAR sampai folder utama terlihat. Jangan langsung membuka project dari file yang masih terkompres.

  2. Buka folder sesuai kebutuhan

    Pilih kategori seperti intro, outro, lower thirds, transitions, text animation, VFX, atau preset. Baca file petunjuk jika tersedia di dalam folder.

  3. Buka project After Effects

    Double click file .aep atau buka After Effects lalu pilih File > Open Project. Tunggu sampai project dan komposisi selesai dimuat.

  4. Import aset tambahan

    Untuk menambahkan logo, foto, video, atau audio sendiri, gunakan File > Import > File atau drag file ke panel Project.

  5. Edit teks dan media

    Buka composition utama, cari layer teks atau placeholder, lalu ganti sesuai kebutuhan. Jika ada placeholder gambar/video, replace dengan footage milik Anda.

  6. Relink file jika missing

    Jika muncul missing footage, cari file yang diminta di folder assets, lalu relink. Usahakan struktur folder tidak dipindah-pindah agar link file tetap aman.

  7. Preview dan render

    Gunakan preview untuk cek animasi. Setelah aman, export melalui Render Queue atau Adobe Media Encoder sesuai format video yang dibutuhkan.

Catatan penting

Jaga struktur folder

After Effects menyimpan referensi ke lokasi file sumber. Jika folder assets dipindahkan setelah project dibuka, beberapa file bisa terbaca missing. Simpan project dan footage dalam satu folder kerja yang rapi.

Mac

Cara menggunakan assets di macOS

  1. Download dan ekstrak file

    Simpan file dari admin Catsoft, lalu ekstrak ZIP/RAR. Jika macOS meminta izin membuka file dari internet, pilih izinkan hanya jika file berasal dari admin Catsoft.

  2. Pindahkan folder ke lokasi tetap

    Simpan folder assets di Documents, Desktop, atau drive project. Jangan membuka project langsung dari folder sementara/download yang akan dipindah.

  3. Buka project .aep

    Buka After Effects, pilih File > Open Project, lalu pilih file .aep dari folder assets.

  4. Izinkan akses file jika diminta

    Jika macOS atau After Effects meminta permission untuk membuka file/folder, pilih Allow agar footage dan assets bisa terbaca.

  5. Edit placeholder

    Buka composition utama, ganti teks, logo, foto, video, atau audio sesuai kebutuhan project.

  6. Preview dan export

    Preview dahulu dengan resolusi Half/Quarter jika berat, lalu export melalui Render Queue atau Adobe Media Encoder.

Catatan macOS

Jangan pindahkan folder setelah dibuka

After Effects menyimpan lokasi file sumber. Jika folder assets dipindah setelah project dibuka, footage bisa menjadi missing dan perlu relink ulang.

Tutorial Relink
Relink Missing

Memperbaiki file yang tidak terbaca

  1. Cari item missing

    Di panel Project, cari item yang memiliki tanda missing atau pesan file not found.

  2. Pilih Replace Footage

    Klik kanan item tersebut, pilih Replace Footage > File, lalu cari file sumber di folder assets.

  3. Pilih file yang sesuai

    Samakan nama file, format, dan foldernya. Jika ada banyak folder, mulai dari folder footage, image, video, audio, atau assets.

  4. Cek item lain

    Jika struktur folder masih sama, After Effects biasanya otomatis menemukan file lain yang berada di folder berdekatan.

  5. Simpan project baru

    Setelah semua file terbaca, gunakan Save As agar project hasil relink tersimpan tanpa merusak file original.

Penyebab umum

Folder dipindahkan

Missing footage biasanya terjadi karena folder assets dipindah, nama file berubah, atau project dibuka sebelum file diekstrak sempurna.

Lanjut Render
Render / Export

Export hasil akhir

  1. Pilih composition final

    Buka composition utama yang sudah diedit. Pastikan durasi, teks, logo, dan media sudah benar.

  2. Cek preview

    Preview area penting terlebih dahulu. Jika berat, turunkan preview resolution ke Half atau Quarter.

  3. Tambahkan ke Render Queue

    Pilih Composition > Add to Render Queue, lalu tentukan output folder dan format yang dibutuhkan.

  4. Atau gunakan Adobe Media Encoder

    Jika ingin format MP4/H.264 lebih mudah, pilih Composition > Add to Adobe Media Encoder Queue.

  5. Render dan cek hasil

    Setelah render selesai, putar video final untuk memastikan tidak ada teks terpotong, media missing, atau warna yang berubah.

Jika render gagal

Cek storage dan plugin

Pastikan ruang penyimpanan cukup, font sudah terpasang, plugin yang dibutuhkan tersedia, dan tidak ada footage missing sebelum render.

Cek Kendala
Troubleshooting

Kendala yang sering terjadi

Project terasa berat

Turunkan preview resolution ke Half atau Quarter, tutup aplikasi lain, dan gunakan preview area yang pendek.

Font berubah

Install font yang disertakan jika ada. Jika tidak tersedia, ganti dengan font lain yang mirip.

Effect tidak muncul

Beberapa template mungkin memakai plugin tertentu. Cek catatan folder atau tanyakan ke admin jika ada error plugin.

Render gagal

Cek ruang penyimpanan, kurangi resolusi preview cache, atau export lewat Adobe Media Encoder.
